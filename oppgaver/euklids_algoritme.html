Referanseimplementasjon

Ved å bruke modulus-operatoren % kan vi finne største felles divisor for to heltall, \(a\) og \(b\) som følger: 

a = int(input("Du vil finne SFD for a = "))  # 27
b = int(input("og b = "))                    # 15

while b != 0:
    temp = b
    b = a % b
    a = temp

print(f"Største felles divisor er {a}.")

Største felles divisor er 3.

Oppgaver

Oppgave 1

Test algoritmen med fem par to-sifrede tall (f.eks. a = 27, b = 15). Skriv ned det du mener er svaret for alle parene før du kjører algoritmen.

Oppgave 2

Test algoritmen med to seks-sifrede tall.

Oppgave 3

Start med tallene a = 27 og b = 15. For disse skal SFD være 3. (Siden \(a = 9\cdot 3\) og \(b = 5 \cdot 3\)).

Skriv deg gjennom hele algoritmen med disse startverdiene. (Hvilke verdier har temp, a og b hver gang gjennom løkka?)

Oppgave 4

Hva skjer dersom du forsøker å regne ut \(\textrm{SFD}(b, a)\) hvor \(b > a\)?

Oppgave 5

Dersom vi ikke ønsker å bruke modulus-operatoren, kan vi finne største felles divisor for to tall ved kun å bruke subraksjon. Vi bruker da at Dersom \(a > b\), så er \(\textrm{SFD}(a, b) = \textrm{SFD}(a - b, b)\).

a

Undersøk om dette stemmer for \(a = 27\) og \(b = 15\).

b

Forsøk å implementere algoritmen uten bruk av modulus-operatoren.
